== Description ==

Style commentators last blog post requires CommentLuv plugin.This Plugin also works with Thesis. 

== Installation ==

To install the plugin follow these steps :

1. Download the style-commentluv.zip file to your local machine.
2. Unzip the file 
3. Upload "style commentluv" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

That's it.

== Frequently Asked Questions ==

How to use this plugin?
